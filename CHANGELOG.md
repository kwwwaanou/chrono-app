# Changelog - Chronosport

## [1.3.2] - 2026-02-18
### Fixed
- **Wake Lock Fix** : Déclenchement du verrouillage d'écran sur geste utilisateur (Play) pour une meilleure compatibilité iOS/Safari.

## [1.3.1] - 2026-02-17
### Added
- **Audio Improvements** : Bip de fin distinct (onde carrée 880Hz).
- **Stability** : Renforcement de la Wake Lock API pour iOS/Safari.

## [1.3.0] - 2026-02-16
### Added
- **Default Theme** : Le mode clair est désormais le thème par défaut.
- **Sets Increment Bug Fix** : Correction de l'incrémentation atomique des séries (plus de doublons).
- **Auto-start** : Le timer se lance immédiatement après la sélection d'une durée.
- **Audio Beeps** : Signal sonore (5s) avec mixage iOS (Web Audio API).
- **Screen Wake Lock** : Maintient l'écran de l'iPhone allumé pendant le timer.

## [1.2.0] - 2026-02-16
### Added
- Mise en place initiale de l'Auto-start, des Bips et du Wake Lock (version de transition).

## [1.1.1] - 2026-02-16
### Fixed
- Polissage de l'interface utilisateur et de la logique interne.
- Restauration du toggle de thème (Sombre/Clair).

## [1.1.0] - 2026-02-16
### Added
- Refonte complète de l'interface : Design "Modern Dark".
- Ajout du pied de page avec versioning.

## [1.0.0] - 2026-02-15
### Added
- Initial commit : Structure de base du projet Chronosport.
