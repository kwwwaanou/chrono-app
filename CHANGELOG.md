# Changelog - Chronosport

## [1.2.0] - 2026-02-16
### Added
- **Auto-start** : Le timer se lance immédiatement après la sélection d'une durée.
- **Audio Beeps** : Signal sonore durant les 5 dernières secondes du compte à rebours.
- **iOS Audio Mixing** : Utilisation de la Web Audio API pour permettre aux bips de jouer par-dessus la musique/YouTube.
- **Screen Wake Lock** : Maintient l'écran de l'iPhone allumé pendant que le timer est actif.
